Connect Claude, Codex, GPT, and other AI agents to WordPress using only WordPress Admin access.

== Description ==

WP Command Center is an AI-powered WordPress Operations Platform: Site Intelligence, Diagnostics, Safe File Patching, Rollback Protection, and an AI Agent Gateway — all from inside wp-admin, with no SSH / WP-CLI / root access required.

See `WP-Command-Center-Canonical-Spec.md` in the plugin root for the full product specification.

== Installation ==

1. Upload the plugin to `/wp-content/plugins/wp-command-center`
2. Activate through the 'Plugins' menu in WordPress
3. Visit "Command Center" in the admin sidebar
