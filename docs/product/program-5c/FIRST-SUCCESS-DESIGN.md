# PROGRAM-5C — Phase D: First Success Experience

## Goal: first value in under 10 minutes
## Honest constraint
True **AI** first-value (SEO/alt-text) cannot be sub-10-minutes for a non-technical, agent-less user because the AI screens are flag-OFF and require an external assistant (F1). Flipping flags / building a human action path is **out of scope** (product/validation work). So 5C delivers the fastest *honest* first value and the clearest path to AI value.

## The redesigned first-success path
1. **~2-minute no-AI win (new):** Overview first-run now leads with a green **"See it work in 2 minutes — no AI or setup needed → Run a site report"** button (read-only Diagnostics). The user gets a real, safe result immediately — building confidence before the harder agent step. *This is the only path that is genuinely sub-10-min for the persona today, and it is honest (read-only, nothing changed).*
2. **Guided setup checklist** (5A/5B): choose Client mode → add AI key (optional) → create token → know where to review/undo, each with live status + links.
3. **Understand the loop** (5B): "How WPCC keeps you in control" (propose → approve → record → undo), now with direct links to Approvals and Changes (Phase E).
4. **Connect the assistant** (5C): the plain-language agent explainer + numbered setup order.
5. **Honest "what next" after a key** (5C): on AI Setup — test the key, connect an assistant, and a candid note that adding a key doesn't enable AI screens by itself.

## Friction reduced
- **Confidence:** a zero-risk win exists before any AI/agent step.
- **Discoverability:** every step links to the next surface; the agent concept is explained at the point of need.
- **Guidance:** ordered path + per-step hints + after-key next steps.
- **Honesty:** no false promise of instant AI; the no-AI win is real and the AI path is clearly signposted (and its gating stated).

## What could NOT be reduced (documented)
- The external-assistant requirement and flag-off AI screens — structural (F1). Mitigated, not removed.

## Validation
`test-first-value-5c.sh` §3 + §5 → no-setup quick win present + read-only + after-key guidance green.

**Phase D: GREEN (best honest first-value path delivered; structural AI gate documented).**
